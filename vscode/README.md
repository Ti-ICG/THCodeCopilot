编写文档很糟糕？让天河国云代码规范助手来处理吧。只需选中所需代码，就能看到神奇的效果。


### 👩‍💻 如何生成文档

**Step 1** 选中代码或将光标放在要记录的行上
**Step 2** 单击“编写文档”按钮(或按⌘+)。
